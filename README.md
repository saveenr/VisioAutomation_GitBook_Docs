# Introduction

## Overview

**VisioAutomation** is a .NET library that simplifies automating **Microsoft Visio**.

**VisioAutomation** helps you:

* Write Visio Add-ins
* Control the Visio application from other applications

**VisioAutomation** is distributed as a NuGet Package:

* **Visio Automation 2010** [NuGet package](http://www.nuget.org/packages/VisioAutomation2010/) (maintained)
* **Visio Automation 2007** [NuGet package](http://www.nuget.org/packages/VisioAutomation2007/) (no longer maintained)
