# Related projects

## Companion module from this project

* [**Visio PowerShell**](https://saveenr.gitbook.io/visiopowershell/) ([source](https://github.com/saveenr/VisioAutomation/tree/master/VisioAutomation_2010/VisioPowerShell)) — a PowerShell module built on top of VisioAutomation. Distributed as the `Visio` module on the [PowerShell Gallery](https://www.powershellgallery.com/packages/Visio).

* [**PowerShell Visio Automation (PSVA)**](https://github.com/jrich523/PSVA) — another PowerShell Visio automation project.
